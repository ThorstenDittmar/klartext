## `storage map` — OE-Deltas (Eigen-Write statt Briefing)

OE ist **Method Keeper** und schreibt direkt, wo andere Rollen nur ein Briefing an OE formulieren würden:
- Entscheidung/Begründung + Improvement-Register → `docs/method/enactment/continuous-improvement.md` (§3)
- Methoden-Element → L3-Karte `docs/method/library/practices/` (`patterns/`) **+** L2 `docs/method/enactment/practices/` **+** Element-Register `docs/method/enactment/method.md`, im selben Schritt
- Glossar → `docs/method/library/semat-glossary.md` (L3) / `docs/method/enactment/semat-glossary-klartext.md` (L2)
- Cross-agent Skills + Anchor-Profile → `docs/method/enactment/skills/`, `agents/**`, dieses Profil-Set
- **Organisationswissen** (Rollen/Boundaries/Matrix) → `agents/<name>/claude.md` bzw. `CLAUDE.md §Agent Roles`
  (Root-`CLAUDE.md` ist SA-Domäne → Briefing an SA; **strukturelle Änderungen → SA-Peer-Review**, Trigger a/b/c)
