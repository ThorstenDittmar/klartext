# 17. Schlussbemerkung: Eine Infrastruktur für verstehbare Komplexität

Die Gegenwart benötigt nicht nur bessere Informationen. Sie benötigt bessere Formen, um Informationen, Annahmen und Kausalitäten zueinander in Beziehung zu setzen.
Das hier beschriebene Projekt schlägt eine solche Form vor: eine Autorenumgebung für epistemisch transparente Narrative.
Sie verbindet wissenschaftliche Modellbildung mit literarischer Erfahrung. Sie nutzt KI nicht als Wahrheitsinstanz, sondern als Werkzeug zur Konsistenzprüfung und Transparenzherstellung. Sie erlaubt konkurrierende Axiomräume, ohne in Beliebigkeit zu fallen. Sie macht sichtbar, welche Annahmen eine Geschichte trägt. Und sie erzeugt medienneutrale Inhalte, die über verschiedene Plattformen verbreitet werden können.
Die langfristige Vision ist eine neue Wissensinfrastruktur:
komplex genug, um reale Debatten ernst zu nehmen;
transparent genug, um Annahmen offenzulegen;
narrativ genug, um Menschen zu erreichen;
strukturiert genug, um maschinell weiterverarbeitet zu werden.
Nicht Wahrheit soll zentralisiert werden.
Verständlichkeit soll infrastrukturell ermöglicht werden.
